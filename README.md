# By The Pragmatic Programmer
# Description
This is a simple MERN  application that holds employees data and allows you to create, update, read or delete their information.

![image](https://user-images.githubusercontent.com/55139857/190352619-7654e3f3-af48-4239-ba30-4be8873d60c2.png)


# User Requirement
The application allows users to do the following:

* Create a new employee.
* Update and existing employees data.
* Delete an employees information
* Display the employees entries.

# Setup/Installation Requirments
* Clone or download and unzip the repository from github,
* If you have git Use this command *git clone*  to clone the project.

## Development server

Run `npm start` for a dev server. Navigate to `http://localhost:3000/`. The app will automatically reload if you change any of the source files.

# Technologies
* React
* Bootstrap
* Express
* MongoDB
* Node.JS


# Behaviour Driven Development
* Input                                           Output
* User fills form and clicks create person      -Employee entry is displayed with input fields for Name, Position and Level
* User Clicks Delete                            -The employee entry is purged from view
* User clicks Edit                              -Employees data is updated with recent data
